# Microsoft Office 365This is a repo for managing machine readable artifacts for the [Microsoft Office 365 APIs](http://office.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Microsoft Office 365 APIs](http://office.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://office.com)- [Developer](http://office.com)- [Twitter](https://twitter.com/OfficeDev)- [GitHub](https://github.com/OfficeDev)The [APIs.json](https://github.com/api-evangelist/microsoft-office-365/blob/master/apis.json) index provides links to all the relevant links for the Microsoft Office 365 APIs, including the OpenAPI and Postman if they are present.
